Employee Management System Using Three Tier Architecture Model

                            This is an employee management software which is built on three tier architecture module, which can perform CRUD methods.
                            
Techniques Used

                            Core Java, JDBC, MySQL, MySQL Connector
                            
IDE Used

                            J2ee IDE 2022-2009
                            
Description

                            This is an employee management software which is built on three tier architecture model. 
                                                             This software performs the CRUD(Create, Read, Update, Delete) Operations. This software 
                                                             also have features like login and logout. By the use of this software we can retrieve all the employee’s 
                                                             data or we can  retrieve a specific employee’s data. The software is also useful in data analysis.

Telephone Directory System Using Three Tier Architecture Model

                    This is a telephone directory software which is built on three tier architecture module, which can perform CRUD methods.
                    
Techniques Used

                    Core Java, JDBC, MySQL, MySQL Connector
                    
IDE Used

                    J2ee IDE 2022-2009
                    
Description

                    This is a Telephone Directory software which is built on three tier architecture model. 
                                            This software performs the CRUD(Create, Read, Update, Delete) Operations. 
                                            This software also have features like login and logout. The software have two interfaces 
                                            one for employees and another for customers. The main feature of the software is data security. 
                                            Only the Boss have access to both employees and customers data, even the even the employees 
                                            cannot access it. The software is also useful in data analysis.
